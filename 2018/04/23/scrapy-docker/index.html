总结

先放干货，使用docker来运行scrapy其实只需要做到以下两点即可：

  1. 在项目的当前目录中
  2. 在scrapy命令前加上docker run --name scrapy --rm -v $(pwd):/runtime/app aciobanu/scrapy

如果你已经会使用scrapy了，看到这里就可以了。
